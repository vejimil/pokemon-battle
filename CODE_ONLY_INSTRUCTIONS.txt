How to use:
1. Extract this ZIP.
2. Copy these files into your existing project root.
3. Keep your current assets/ folder exactly as it is.
4. Replace files when prompted.
5. For the migrated local singles engine path, run `npm start` and open `http://localhost:4173`.
6. If you do not run the local server, the app will still fall back to the old custom browser battle runtime.
